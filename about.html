About the Project

"From 1 TO 5 EN" is a mini-website created by a high-school student to help beginners learn English with tiny lessons and instant quizzes. The project is fully static (HTML/CSS/JS) and free to use.

Goals

  • Make English practice quick and friendly.
  • Offer short quizzes with immediate feedback.
  • Keep everything simple so it works on any device.

Built with ♥ using HTML, CSS, and JavaScript.
